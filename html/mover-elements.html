Mover Elementos


Sakura Card Captor ha implementado la clase _movimiento-elements es una clase de estilo definida en un archivo SASS que se utiliza para aplicar propiedades de posicionamiento, alineación y justificación a elementos. Esta clase utiliza varios mixins para establecer propiedades como posición (con opciones para relative, absolute, fixed, etc.), alineación vertical y horizontal, justificación. La clase proporciona una manera fácil de aplicar estas propiedades de estilo de manera coherente en diferentes elementos, lo que ayuda a mantener un diseño consistente y bien estructurado. 


                    
                        _mover-elements.scss

                        @mixin position($propiedad, $top,$left,$right,$boton){
                            position: $propiedad;
                            top: $top;
                            left: $left;
                            right:$right;
                            bottom: $boton;  
                        }
                
                        @mixin alineacion($align-items,$align-self,$align-content){
                            align-items:$align-items;
                            align-self: $align-self;
                            align-content:$align-content;
                        }
                        
                        @mixin justificar($justify-content,$justify-items, $justify-self){
                            justify-content: $justify-content;
                            justify-items: $justify-items;
                            justify-self: $justify-self;
                        }
                        
                        @mixin movimiento($move){
                            float:$move;
                        }
