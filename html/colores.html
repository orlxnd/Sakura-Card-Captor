Colores


La clase colores es una clase de estilo diseñada para el sitio web de Sakura, que cuenta con una variedad de colores vibrantes y atractivos inspirados en la estética de la serie Sakura Card Captor. Esta clase se utiliza para aplicar estos colores específicos a diferentes elementos y componentes del sitio web, lo que contribuye a crear una experiencia visual encantadora y cohesiva para los visitantes. 


                    
                        _colores.scss

                        //Colores Sakura
                        $color_Sakura-dress1: #F2AEC7;
                        $color_Sakura-ayes: #009572;
                        $color_Sakura-dress2:#D17C9B;

                        $color_Sakura-Wand: #FFC366;
