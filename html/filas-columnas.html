Filas y Columnas


Sakura Card Captor tiene un sistema de filas y columnas mediante containers de tipo grid. Por lo cual nuesta libreria tiene un archivo llamado _columnas-filas-grid.scss donde almacena funciones que ayudan a definir las columnas y filas. Nuestra clase _columnas-filas-grid.scss proporciona sus funciones la clase _grid.scss . 


                  
                    _columnas-filas-grid.scss
    
                    @mixin col($col){
                      grid-template-columns: $col;
                    }
                    @mixin row($row){
                        grid-template-rows: $row;
                    }


Considerando la conexión entre los archivos _columnas-filas-grid.scss y _grid.scss en Sakura Card Captor, encontramos que _grid.scss alberga las clases esenciales para definir columnas y filas. Estas clases, sin embargo, son el resultado de las funciones específicas contenidas en _columnas-filas-grid.scss. Sakura Card Captor ha fijado ciertos límites en cuanto al número máximo y mínimo de columnas y filas permitidas en un diseño: hasta 4 columnas y al menos 1, lo mismo aplicado a las filas. Aunque, estas clases pueden ser ajustadas según las necesidades en sus respectivos archivos. Esto brinda la flexibilidad de modificar las dimensiones tanto de columnas como de filas, ya que la biblioteca establece un tamaño predeterminado de 1fr para cada una.


Clases Columnas Clases Filas
.-col1 .-row1
.-col2 .-row2
.-col3 .-row3
.-col4 .-row4

Sakura Card Captor también ha establecido funciones para que un elemento de tipo grid ocupe un área dentro de la cuadrícula. Asi mismo, incluye una función para el espaciado entre filas y columnas. Estas funciones pueden ser modificadas dentro del archivo _columnas-filas-grid.css. 


                
                  Archivo: _columnas-filas-grid.scss
    
                  @mixin espaciado($col,$row){
                    column-gap: $col;
                    row-gap: $row;
                  }
                  @mixin area-col($inicio,$final){
                      grid-column-start: $inicio;
                      grid-column-end: $final;
                  }
                  @mixin area-row($inicio,$final){
                      grid-row-start: $inicio;
                      grid-row-end: $final;
                  }

Para definir un contenedor como grid, en Sakura Card Captor se ha creado una clase llamada -grid, la cual se encuentra en el archivo _grid.scss.


                  
                    Archivo _grid.SCSS
    
                    .-grid {
                        display: grid;
                      }
