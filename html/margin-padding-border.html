Margins Paddings Border


Sakura Card Captor tambien ha implementado la clase _margin-padding-border es una clase de estilo definida en un archivo SASS que utiliza mixins para agregar propiedades de margen, relleno y borde a elementos del sitio web. Esta clase utiliza tres mixins: padding, margin y border, que permiten establecer fácilmente el relleno, el margen y el borde de un elemento, respectivamente. 


                    
                        _margin-padding-border.scss

                        @mixin padding($padding){
                            padding: $padding;
                        }
                        @mixin margin($margin){
                            margin: $margin;
                        }
                        @mixin border($border){
                            border: $border;
                        }
